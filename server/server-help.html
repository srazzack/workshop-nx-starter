Nx Enterprise Workshop - REST Server

REST API documentation for custom HTTP/REST server.

Lookup API EndPoint URL

  • getAllUsers() /api/users GET
  • getAllTickets() /api/tickets GET
  • getAllCompanies() /api/companies GET
  • getAllEvents() /api/eventlogs GET
  • getAllComments() /api/comments GET
  • getTicketById(4) /api/tickets/:id GET
  • getTicketComments(5) /api/tickets/:id/comments GET
  • getUserById(8) /api/users/:id GET
  • getUsersByName('je') /api/users?searchTerm=<val> GET
  • getCompanyById(3) /api/companies/:id GET
  • getUsersAtCompany(3) /api/companies/:id/users GET
